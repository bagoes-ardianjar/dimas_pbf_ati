POS Product Switch , List view With Sorting Odoo Apps

POS Product Switch View and Sort Odoo App helps users to switch point of sale products between grid view and list view. Users can easily sort the point of sale product list by name, code, UOM, sale price and available quantity.

Features

POS Product Switch between Grid View and List View

User can switch point of sale product between grid view and list view.

Easy to Sort POS Product

User can easily sort point of sale product by name, code, UOM, sale price and available quantity.

Display Different Product Field Option

User have option to display different product field like code, type, uom, price, on hand and forecast qty.

Display Product Image with Different Size

User have option to display product image in list view with different option like, small image, medium image and large image.
Demo Request Video Tutorial

Product Switch View Configuration

Under Point of Sale > Configuration > Point of Sale menu user have option to switch product view in list or grid view. User also have different option to view in tree view like product name, type, reference code, uom, price, on hand qty, Forecast qty, and show image in different view like small, medium, and large.

POS Product Grid View

User can easily switch point of sale product in grid view and list view by clicking icon as shown image.

POS Product List View with Small Image

User have to select "List View" as "Default Product View" and select "Small Size" option.

POS Product List View with Medium Image

User have to select "List View" as "Default Product View" and select "Medium Size" option.

POS Product List View with Large Image

User have to select "List View" as "Default Product View" and select "Large Size" option.

POS Product List View Option

User can only see selected option in list view.

Search POS Product in List View

User can search point of sale product in list view and see the results.

Sort POS Product by Code

User can easily sort POS product list by code, name, UOM, sale price or available quantity.
